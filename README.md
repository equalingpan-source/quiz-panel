# Quiz Panel Handwriting

`quiz-panel` を複製して作った、手書き回答対応版です。

## 追加したこと

- 親機で回答方法を選び、回答受付を開始可能
- 手書き回答はキャンバスに描いてそのまま送信可能
- 司会画面とモニター画面でも手書き回答を画像として表示
- 元の `quiz-panel` フォルダは未変更

## フォルダ

- 元版: `C:\Users\b1147\マイドライブ\31.Webアプリ_完成品置き場\quiz-panel`
- 手書き版: `C:\Users\b1147\マイドライブ\31.Webアプリ_完成品置き場\quiz-panel-handwriting`

## 起動方法

```bash
npm install
npm start
```

起動後の司会画面:

```text
http://localhost:3000/host.html
```

## プレイヤー操作

1. 司会がルームを作成
2. 参加者が QR または参加 URL から入室
3. 親機で今回の回答方法を選択
4. 親機が `回答受付を開始` すると、プレイヤーが入力可能
5. 回答後に送信

## 補足

- 手書き回答は PNG データとしてサーバーに送っています
- 入力停止時の自動ロックでも、選択中のモードの内容が保存されます
