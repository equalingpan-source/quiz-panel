# Quiz Panel

親機と子機をブラウザでつないで使う、リアルタイム回答表示用のクイズアプリです。
モニター画面に回答カードを並べて表示できるので、会場投影や配信画面にも使えます。

## できること
- 親機でルームを作成
- QRコードまたはURLから子機を参加
- 子機から回答を送信
- 親機で回答表示、正解判定、次の問題への進行
- モニター画面に回答カードを一覧表示

## かんたんな起動方法

### 1. Node.js を入れる
Node.js 18 以上を入れてください。

### 2. 起動する
いちばん簡単なのは `start_quiz_panel.bat` をダブルクリックする方法です。

このバッチは次のことを自動で行います。
- Node.js の存在確認
- 必要な依存関係のインストール
- 親機画面の起動
- サーバー起動

### 3. 手動で起動する場合
```bash
npm install
npm start
```

起動後、ブラウザで次を開きます。
```text
http://localhost:3000/host.html
```

同じネットワーク内の別端末から参加する場合は、親機画面に表示されるURLやQRコードを使ってください。

## 画面
- `/host.html` : 親機
- `/player-entry.html` : 子機の参加画面
- `/player.html` : 子機
- `/monitor.html` : モニター

## Render で公開する
GitHub にこのフォルダを push して、Render の Web Service としてデプロイします。

設定の目安は次の通りです。
- Runtime : `Node`
- Build Command : `npm install`
- Start Command : `npm start`

デプロイ後は、発行されたURLにアクセスすればそのまま使えます。
公開URLで開いた親機は、そのURLを基準に子機用URLとモニターURLを出すようになっています。

ヘルスチェックを使う場合は次を指定できます。
```text
/healthz
```

## 配布時に一緒に渡すもの
- `start_quiz_panel.bat`
- `reinstall_dependencies.bat`
- このフォルダ一式

## 補足
- 親機をリロードしても、一定時間内なら同じルームに復帰できます。
- うまく起動しないときは `reinstall_dependencies.bat` を実行して依存関係を入れ直してください。
- サーバーを止めるときは、起動した黒い画面を閉じてください。
- Render の無料プランでは、しばらく使わないとスリープ復帰で最初の表示が少し遅くなることがあります。
